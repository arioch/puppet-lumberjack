Example usage

Install lumberjack

node /box/ {
  include lumberjack
}

Configure lumberjack

node /box/ {
  class { 'lumberjack':
    host        => '127.0.0.1',
    port        => 5000,
    window_size => 1024,
    logs        => [ '/var/log/*', '/var/log/*/*' ],
  }
}
